Projects

Parallax A visual effect of different elements moving at various rates. The effect can be used to create depth and interaction to illustrations - a great way to engage the viewers and personalizing their experience, as they control the movement by scrolling.


Responsive: A versatile approach in crafting layouts and user experiences for screen-based media, ensuring seamless adaptation to the user's specific format. Tailoring content to suit varying screen sizes optimizes the presentation of different types of information, thereby enhancing the overall user experience.


Unit converter: A JavaScript based project that use user-input to create interaction and a dialogue between the element and the user. This is done through a unit conversion element, where user input is changed and returned.


To-Do-List: A generator for a To-Do-List that combines HTML, CSS and JavaScript. Based on user-input new HTML elements are created in a list, which can be toggeled between states of completion and removed when done.
